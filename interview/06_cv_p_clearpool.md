## solidity
```
Developed core lending protocol contracts (pool creation, lending, staking, settlement, liquidation)
Designed and implemented a gas-efficient Merkle-tree-based whitelisting mechanism to reduce sybil and unauthorized access risks.
Designed and implemented multi-signature wallet, time-lock, and emergency pause mechanisms to enhance asset safety
```

## golang
```
Developed an event collection system based on go-ethereum to collect on-chain events (pool creation, lending, staking, settlement, and liquidation modules) in real time
Developed  the API services using Golang (Redis caching, db read-write separation, db hot/cold data separation )

```

## data
