# 借贷协议主要方法
```
借贷协议主要方法 ：
poolCreation：创建流动性池子（Pool）
lending / supply：出借资产到协议，增加流动性，获得利息。
redeem：出借方赎回存款和利息 。
staking：质押资产到协议或流动性池子。
borrow：借款方从协议借入资产。
repayBorrow：借款方归还借款，包括本金和利息。
liquidation（清算）：当借款方抵押率低于阈值时，协议触发清算。
```
