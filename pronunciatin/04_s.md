## 复数s的读法
```
1. 发 /s/（清音）
当前一个音是清辅音（除了 /s/, /ʃ/, /tʃ/ 等特殊情况）时，-s 发 /s/。
例子：
cats /kæts/
books /bʊks/
maps /mæps/

2. 发 /z/（浊音）
当前一个音是元音 或 浊辅音（除了 /z/, /ʒ/, /dʒ/ 等特殊情况）时，-s 发 /z/。
例子：
dogs /dɑːɡz/
bags /bæɡz/
cars /kɑːrz/
tables /ˈteɪbəlz/
```

## 清音/浊音 
```
发音器官区别
浊辅音 (voiced)：声带震动（把手指放在喉咙能感觉到嗡嗡的震动）。
清辅音 (voiceless)：声带不震动，只靠气流发声。
清辅音	浊辅音	示例单词
/p/ 	/b/  	pat vs bat
/t/ 	/d/ 	ten vs den
/k/ 	/ɡ/ 	cap vs gap
/f/ 	/v/ 	fan vs van
/s/ 	/z/ 	sip vs zip
/ʃ/ 	/ʒ/ 	ship vs garage
/tʃ/    /dʒ/ 	cheap vs jeep

```
