##  常见数据结构
```
一、数组与字符串
二、链表
三、栈与队列
栈（Stack） = 后进先出（LIFO）
队列（Queue） = 先进先出（FIFO）
栈内存 （Stack）：后进先出
堆内存 （heap）: 随机访问
四、哈希表
五、树与二叉搜索树（BST）
1. mysql 的索引为什么用 b+tree
2. 讲一下 merkel tree

七、排序与查找

冒泡排序 ：每次遍历数组，把相邻的元素两两比较，把大的冒泡到末尾
 对比次数少，交换次数多
选择排序：每次从未排序部分选择最大的元素，放到已排序部分末尾
 对比次数多，交换次数少

```
